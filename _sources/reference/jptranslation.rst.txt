
このドキュメントについて
========================

この文書は、Pythonドキュメント翻訳プロジェクトによる Python Language Reference
の日本語訳版です。日本語訳に対する質問や提案などがありましたら、Pythonドキュメント翻訳プロジェクトのGoogle Group

`<https://groups.google.com/forum/#!forum/python-doc-jp>`_

または、GitHub 上の Issue Tracker

`<https://github.com/python-doc-ja/python-doc-ja/issues>`_

までご報告ください。
