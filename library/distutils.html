27.1. distutils — Python モジュールの構築とインストール¶

distutils パッケージは、現在インストールされている Python に追加するためのモジュール構築、および実際のインストールを支援します。新規のモジュールは 100%-pure Python でも、C で書かれた拡張モジュールでも、あるいは Python と C 両方のコードが入っているモジュールからなる Python パッケージでもかまいません。

このパッケージについては2つに分割された章で議論されています。

参考

Python モジュールの配布

Python モジュールの開発者およびパッケージング担当のためのマニュアルです。この文書では、既にインストールされている Python に、 distutils を使ったパッケージをより簡単にインストールできるような、準備の方法について解説しています。

Python モジュールのインストール

現在インストールされている Python にモジュールをインストールするための情報を含む “管理者” マニュアルです。このマニュアルを読むのに、あなたが Python プログラマである必要はありません。
